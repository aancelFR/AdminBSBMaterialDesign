Mise à jour du template Admin : https://github.com/gurayyarar/AdminBSBMaterialDesign

Utilisation de Vue-Cli4
Le template principal a été implémenté sous forme de component avec VueJs.
Le style a été mis avec des fichiers SCSS.
Des parties de fonctionnalité ont été également récupérées sur le template AdminLTE.



Installer les dépendances avec npm install et npm run install (pour les dépendances réalisées avec napa pour les dépôts qui n'était alors pas installable par npm)

Pour lancer le projet en mod dev:

npm run serve
